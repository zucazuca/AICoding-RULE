# Templates

本目录保存按需复制的结构化交接模板，不进入项目默认 Required Reading。

## Three-Authority VibeCoding Governance

完整工作流见[工作流入口](../workflows/three-authority-vibecoding/README.md)，角色指令见[Prompt 索引](../prompts/three-authority-vibecoding/README.md)。

| 模板 | 用途 |
|---|---|
| [execution-package](three-authority-vibecoding/execution-package.md) | 审批窗口向执行窗口下发施工包 |
| [execution-report](three-authority-vibecoding/execution-report.md) | 执行窗口回传候选提交与自测证据 |
| [test-request](three-authority-vibecoding/test-request.md) | 审批窗口向测试窗口下发冻结验收包 |
| [independent-test-report](three-authority-vibecoding/independent-test-report.md) | 测试窗口回传独立验收证据 |
| [approval-record](three-authority-vibecoding/approval-record.md) | 审批窗口记录代码审查、测试、推送、发布和 Owner 决策 |
