# Three-Authority VibeCoding Templates

按状态机顺序使用：

1. [execution-package](execution-package.md)
2. [execution-report](execution-report.md)
3. [test-request](test-request.md)
4. [independent-test-report](independent-test-report.md)
5. [approval-record](approval-record.md)

所有候选阶段模板必须填写同一个完整 Candidate-Commit。空字段写 NONE 并说明原因，不得删除字段或用模糊引用代替。
